## Guess Game Application

**Guess Game By Bhumi Jain**

A simple Java-based game where players guess a randomly generated number. The program provides hints like **"too high"** or **"too low"** and displays the total number of attempts when the correct number is guessed. This project is perfect for practicing Java basics such as loops and conditionals.

1. The program generates a random number.
2. The user enters a guess.
3. The program checks the guess and prints:
